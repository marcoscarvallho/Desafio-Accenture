# Desafio API - Sistema de Reserva de Livros

## Descrição
Automação de testes de API usando Python com Behave (BDD) e Page Object Pattern.

## Tecnologias
- Python 3.x
- Behave (BDD)
- Requests
- Page Object Pattern

## Como Executar

### Execução com BDD (Behave)
```bash
pip install -r requirements.txt
behave -v
```
