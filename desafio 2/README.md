# Desafio Web - Automação DemoQA

## Descrição
Automação de testes web usando Python com Behave (BDD) e Page Object Pattern.

## Tecnologias
- Python 3.x
- Selenium WebDriver
- Behave (BDD)
- Page Object Pattern

## Como Executar

### Execução com BDD (Behave)
```bash
pip install -r requirements.txt
behave features/
```
